# データの持ち方、表示の仕方

トップページには、それぞれのカテゴリから最新 3 つ程度の記事を出す

## サイトカテゴリ

- code(Github Project)
- 同人
- Writing
- Sports
- Owned blog/website

## code

Github のデータに直接クエリ

## 同人, Writing, Sports

Markdown の Article にタグをつけて、それぞれのタグ最新でクエリ

## Owned blog/website

サイト設定か、json ファイルから読み出して記事とは別の場所にリストで載せる

## その他

岸田教団のレポートなど、他サイトを紹介しないといけないので OGP カードを作る拡張を入れておく
